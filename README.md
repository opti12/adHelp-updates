# adHelp 애플리케이션

**Active Directory (AD) 관리 및 헬프데스크 지원을 위한 유틸리티 프로그램**

`adHelp`는 AD 사용자 계정 잠금 해제, 비밀번호 재설정, 그룹 조회 등 빈번한 헬프데스크 업무를 간소화하고, 보안을 유지하면서 관리 도구(`ClientDevices.msc`)를 쉽게 실행할 수 있도록 돕는 Windows 데스크톱 애플리케이션입니다.

## 주요 기능

### 1. 사용자 계정 관리
*   **사용자 검색**: ID (eMail ID)으로 AD 사용자를 신속하게 검색합니다.
*   **상세 정보 조회**: 이름, 부서, 이메일, 계정 상태(활성/잠금/만료) 등을 한눈에 확인합니다.
*   **계정 잠금 해제 (Unlock)**: 잠긴 계정을 즉시 해제합니다.
*   **비밀번호 재설정 (Password Reset)**: 도메인 정책에 맞는 안전한 새 비밀번호로 재설정하거나, 임의의 강력한 비밀번호를 생성하여 적용합니다. (다음 로그인 시 변경 옵션 지원)
*   **자격 증명 검증**: 사용자의 현재 비밀번호가 유효한지 테스트할 수 있습니다.

### 2. 그룹 관리
*   **그룹 멤버십 조회**: 특정 사용자가 소속된 모든 AD 그룹 목록을 확인합니다.
*   **클립보드 복사**: 그룹 목록을 텍스트로 복사하여 문서화나 티켓 처리에 활용할 수 있습니다.

### 3. 시스템 관리 도구 연동
*   **Client Devices 관리 (`ClientDevices.msc`)**:
    *   프로그램에 내장된 `ClientDevices.msc` 도구를 관리자 권한으로 실행합니다.

### 4. 보안
*   **안전한 인증**: 비밀번호는 메모리 내에서만 안전하게 처리됩니다.

## 시스템 요구사항

*   **OS**: Windows 10/11 또는 Windows Server
*   **Framework**: .NET Framework 4.7.2 이상
*   **네트워크**: Active Directory 도메인 컨트롤러와 통신 가능한 환경

## 설치 및 실행

1.  배포된 `adHelp.exe` 파일을 다운로드합니다.
2.  프로그램을 실행하고, 필요한 경우 AD 관리 권한이 있는 계정으로 로그인합니다 (프로그램 내부 로그인 기능).
3.  자동 업데이트 기능이 내장되어 있어, 새로운 버전이 배포되면 실행 시 알림을 받습니다.
